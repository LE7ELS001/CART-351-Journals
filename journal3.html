This week, we focused on Flask, a web framework written in Python. Unlike the static approach of writing HTML with Java in the past, Flask allows web pages to dynamically display different results based on user input and retrieve data from a database to render on the page. In Flask, the code app = Flask(__name__) means that we are turning the current file into a web application, and we use the variable app to reference it. And some code like: @app.route('/') def xxx(): return render_template("xxx.html") app.run() This allows us to use Flask to serve the corresponding HTML page and display it in the browser. We also learned some Jinja2 syntax. For instance: def xxx(): userChoice = False return render_template("a.html", answer = userChoice) In the corresponding a.html, if we have: {% if answer %} p The answer is Yes p {% endif %} then the browser will display “The answer is Yes.”
